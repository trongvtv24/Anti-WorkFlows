# Prompt Template — M7: Số liệu / Cập nhật mới nhất (VN 2025-2026)

Mảnh này cover gap mà AI Overview thường không có vì data quá mới.

---

```
// ROLE
Bạn là [chuyên gia] đang cập nhật tình hình mới nhất cho [audience].

// TASK
Dựa CHÍNH XÁC và CHỈ trên dữ liệu tôi cung cấp, viết phần SỐ LIỆU / CẬP NHẬT:
**[Tên H2 của mảnh M7]**

// CONSTRAINTS
- Độ dài: [150-300] từ (tùy lượng data có được)
- Chỉ viết nếu có đủ data Tier 1-2 với ngày publish rõ ràng
- Ghi rõ nguồn và năm cho MỌI số liệu cụ thể
- Format: Mixed — số liệu inline trong đoạn văn + highlight những thay đổi quan trọng
- Nếu data cũ (>2 năm): ghi chú rõ "theo dữ liệu [năm]" và khuyến nghị verify
- Gap phải cover: [Thường là số liệu địa phương VN / thay đổi chính sách / sản phẩm mới]

// TUYỆT ĐỐI KHÔNG:
- Extrapolate số liệu ("có thể lên đến...", "dự kiến...")
- Dùng số liệu mà không có nguồn rõ ràng
- Viết về thay đổi chính sách/pháp lý nếu không có Tier 1 confirm

// BLACKLIST (chuẩn)
[Các pattern blacklist tiêu chuẩn]

// DATA INPUT
[Paste raw data số liệu từ G4 — bắt buộc có ngày publish]
Nguồn: [tên] | Tier [1/2/3] | Ngày: [dd/mm/yyyy]

// OUTPUT
1. N số liệu cụ thể đã cite (đếm)
2. Nguồn mới nhất là từ năm nào?
3. Có số liệu nào cần verify thêm không?
4. Gap coverage
```

---

## Lưu ý đặc biệt

**Nếu không tìm được đủ data mới tại G4:**
→ Bỏ mảnh M7 khỏi outline
→ Không nhét số liệu cũ / không đáng tin vào chỉ để "có section"
→ Ghi note trong Verify List: "Chưa tìm được số liệu VN 2025-2026 — user cần verify thêm"

**Format cite số liệu trong bài:**
```
Theo [tên tổ chức], năm [YYYY], tỷ lệ [X] đạt [Y%] tại [địa điểm].
```
Không viết kiểu: "Số liệu cho thấy..." mà không có nguồn.
